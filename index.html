Proyectos (Versión Estática)

Realicé un curso de Microsoft Office hace unos años para mejorar mis habilidades con los diferentes programas que contiene el paquete Office. Fue un curso de un año que lo realicé en 2021 mientras acababa 2º de Bachillerato, me sirvió bastante para profesionalizarme con PowerPoint, Excel o Word y así hacer entregas diferentes y originales que los profesores valoraron bastante.

Actualmente nos encontramos en clase aprendiendo Bootstrap, muy útil para desarrollar páginas web y mucho más cómodo que css al tener ya unos códigos predefinidos para hacer lo que necesites.

Curso realizado durante el ciclo de 1º de ASIR, que me sirvió para entender y aprender a manejar los dispositivos de red de una manera virtualizada. Muy recomendable para la gente que vaya a estudiar ASIR.
